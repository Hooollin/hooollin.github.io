markdown基本语法回顾

发表于 2020-01-01 更新于 2020-11-23 分类于 日常学习

一、标题

markdown中用连续的若干个放置在文字之前的’#’来表示标题，一个’#’代表一级标题，两个’#’代表二级标题，以此类推。支持六级标题。

二、字体

加粗

要加粗的文字两边用两个*包起来。需要在同一行才会生效，否则只是普通字符。

如：加粗。

斜体

要倾斜的文字左右分别用一个*号包起来。同样需要在同一行内才会生效。

如：斜体。

斜体加粗

要倾斜和加粗的文字左右分别用三个*号包起来。需要在同一行内才会生效。

如：**斜体加粗**。

删除线

要加删除线的文字左右分别用两个~~号包起来。需要在同一行内才会生效。

如：删除线

六、列表

无序列表

无序列表用 - + * 任何一种都可以。

如：

  • 第一项
  • 第二项
  • 第三项

    有序列表

    数字加点。序号和内容之间有空格。 如：
  1. 第一
  2. 第二
  3. 第三

列表嵌套

  • 第一项
    1. 第一项第一点
    2. 第一项第二点
  • 第二项
    1. 第二项第一点
    2. 第二项第二点
