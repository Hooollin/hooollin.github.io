四种基本数据类型

数据类型 类型关键字 长度 位数 范围
字符型 char 1字节 8 -128~127
整型 int 2字节 32 $-2^{31}$ ~ $2^{31} - 1$
单精度浮点型 float 4字节 32 $\pm(10^{-38}$ ~ $10^{38})$
双精度浮点型 double 8字节 64 $\pm(10^{-308}$ ~ $10^{308})$

C语言没有具体规定整型数据所占内存的字节数，一般来说，$2=short \leq int\leq long=4$。
字符型的数据是根据其对应的ASCII码存储，所以字符型数据可以看做成整型数据，也可以表示为unsigned char，范围从0~255，此时ASCII大于127的字符为扩充的ASCII码字符。
